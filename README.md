# tfg-coledocolitiasis
# Desarrollo y validación de modelos predictivos basados en aprendizaje automático para la estimación del riesgo de coledocolitiasis en pacientes con sospecha clínica
Este repositorio contiene el código fuente desarrollado para el Trabajo de Fin de Grado (TFG) orientado a la optimización del diagnóstico y predicción del riesgo de coledocolitiasis utilizando algoritmos de Machine Learning.

## Contenido del Repositorio
*   `EDA_TFG.ipynb`: Script correspondiente al análisis exploratorio de datos (EDA)
*   `Coledocolitiasis.ipynb`: Código relativo a los modelos
